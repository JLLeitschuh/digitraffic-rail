Ennusteiden lähteet

Suomessa käytetään kahta eri järjestelmää ennusteiden tuottamiseen: MIKU:a ja LIIKE:ttä. rata.digitraffic.fi:ssä näiden kahden järjestelmän ennusteita pyritään yhdistämään yhteen JSON-kenttään liveEstimateTime.

Järjestelmien välillä on eroja:

  • MIKU toimii pääasiassa Etelä-Suomessa ja Liike muualla. MIKU-ennusteet ovat arvokkaita erityisesti Etelä-Suomen junien häiriötilanteissa.
  • MIKU-ennusteet ovat liikennepaikkojen laiturinäytöillä käytettyjä ennusteita, LIIKE-ennusteet ovat pääasiassa liikenteenohjaajien tuottamia ennusteita.
  • rata.digitraffic.fi:ssä MIKU-järjestelmästä hyödynnetään vain käsin tehtyjä ennusteita. LIIKE-järjestelmästä hyödynnetään myös toteumien pohjalta lasketut automaattiennusteet.

estimateSource-kenttä kertoo mistä ennuste on peräisin. Kentän arvot voivat olla

  • MIKU_USER
  • LIIKE_USER
  • LIIKE_AUTOMATIC
  • COMBOCALC

MIKU_USER

MIKU-järjestelmässä tehty käsiennuste (eli ihmisen tekemä ennuste).

LIIKE_USER

LIIKE-järjestelmässä tehty käsiennuste.

LIIKE_AUTOMATIC

LIIKE-järjestelmän tekemä automaattiennuste, joka on laskettu edeltävän liikennepaikan ennusteen tai toteuman pohjalta. Liikkeen automaattiennusteet olettavat useimmiten junan kirivän aikatauluansa kiinni.

COMBOCALC

LIIKE-järjestelmän tekemä automaattiennuste, joka on muodstettu yhdistämällä LIIKE- ja MIKU-järjestelmien ennusteita.
